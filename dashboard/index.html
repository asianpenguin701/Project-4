From Takeoff to Arrival: Predicting Weather-Related Delays with Machine Learning

About the Model

This dashboard utilizes a Random Forest machine learning model to predict weather-related flight delays across various airports in the United States. The model takes into account real-time weather data such as temperature, precipitation and wind speed to estimate the probability of a delay at a given airport.
